## About The Project

This project is built to practice how to connect between spring boot and mongodb

### Built With

* [Java](https://www.java.com/en/)
* [Spring](https://spring.io/)

### Tested With
* [Postman](https://www.postman.com)

### Prerequisite
* JDK 17
* Java 8 - lambda expression
* MongoDB
* Postman
